# About
An ugly but working mining recorder for Doraemon farming game of Nintendo Switch

# Init project
```
yarn install
```

# Development
```
yarn vue serve
```

# Build
```
yarn vue build
```

# Note
- could set mine by keyboard, but not well tested, probably Chrome only
- mobile friendly
